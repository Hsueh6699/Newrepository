簡介

ONEPAY 是一個基於 Telegram 平台的智能支付機器人，為用戶提供安全、便捷的充值和提款服務。它支持多種支付方式，包括銀行卡轉賬和支付寶支付，並提供實時訂單查詢功能。

功能特點

  • 支持多種支付方式：銀行卡轉賬、支付寶等
  • 實時訂單查詢：隨時掌握交易狀態
  • 商戶資訊查詢：快速了解賬戶餘額
  • 靈活的授權管理：確保只有授權用戶可以使用
  • 簡單直觀的操作流程：易於使用，無需專業知識
  • 支持多達69家銀行：滿足各種用戶需求
  • 隨時新增或刪除充值方式和提款銀行：靈活管理支付選項

安全性

ONEPAY 採用多重安全措施保護用戶資金和信息安全:

  • 加密通信：所有數據傳輸均經過加密，確保信息不被竊取
  • 授權機制：只有經過授權的用戶才能使用機器人，防止未授權訪問
  • 簽名驗證：每次交易都進行簽名驗證，確保交易的真實性和完整性
  • 敏感信息保護：不存儲用戶的銀行卡號等敏感信息，降低信息洩露風險
  • 安全的API調用：使用HTTPS協議進行API調用，確保數據傳輸安全
  • 定期安全審計：定期進行安全審計，及時發現和修復潛在的安全漏洞
  • 機器人擁有者控制：只有機器人擁有者可以啟用並授權給指定用戶使用

使用教程

1. 機器人設置

將機器人添加到 Telegram 群組，並由管理員使用 /authorize 命令授權用戶。

2. 充值流程

  1. 使用 /deposit 命令開始充值
  2. 輸入充值金額
  3. 輸入打款人姓名
  4. 從提供的選項中選擇支付方式
  5. 按照機器人提供的指示完成支付

3. 提款流程

  1. 使用 /withdraw 命令開始提款
  2. 選擇提款方式（銀行卡或支付寶）
  3. 如選擇銀行卡，輸入銀行名稱或選擇銀行
  4. 輸入提款金額和相關信息（如賬號、姓名等）
  5. 確認提款信息

4. 查詢功能

使用 /query 命令查詢訂單狀態，輸入商戶訂單號即可獲取詳細信息。
使用 /merchant_info 命令查詢商戶資訊，包括可用餘額、凍結餘額和總餘額。
